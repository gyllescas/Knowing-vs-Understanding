Gabriel Yllescas's CS601 Concept

After spending weeks working with a struggling 2nd grade student with common core math I have realized that visualizing the base 10 number system is something that kids really struggle with. I hope to build a learning tool to help visualize basic arithmetic operations + , -, *, /

Aproach

  • Parent enters homework problems.
  • A helpful character enters the application ready to walk the student through solving each problem
    • The base 10 number system is usually represented with blocks inside class rooms, but on homework assignments blocks are drawn in in 2 dimensions. I will find a javascript library to create 3d groups of blocks representing the one's, ten's, and hundred's place
  • The student will set up the problem representing each number using blocks. For example 42 + 10 is 4 ten's and 2 one's + one ten. Only after completing this step will he be able to move on to solving the problem.
  • A goal will also be for the student to be able to enter common core word problems. For example "What is one ten more than 4 tens and 2 one's"
      This part of the project might be extremely difficult. Writing a word paraser from scratch might be a stretch of a task. I will search for a javascript or a serverside php library to accomplish the parsing.
  • A counter will keep track of the amount of time spent on the application and the success rate for each problem entered.
  • On the server, logs of each session will be kept for teacher/parents to pin point exactly where a student struggles.
